## â–¶ï¸ Getting Started

### Prerequisites

* .NET 9 SDK
* Visual Studio / VS Code
* SQL Server or SQLite

### Clone the Repository

```bash
git clone https://github.com/your-username/dotNet9API_Preactice.git
cd dotNet9API_Preactice
```

### Run the Application

```bash
dotnet restore
dotnet run
```

The API will be available at:

```
https://localhost:5001
```

Swagger UI:

```
https://localhost:5001/swagger
```
